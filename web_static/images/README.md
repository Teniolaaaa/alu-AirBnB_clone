# Images Directory

This directory should contain the following image files:

## Required Images:

1. **icon.png** - Browser tab icon (favicon)
   - Used in the browser tab for the website

2. **logo.png** - Website logo
   - Dimensions: 142px width x 60px height
   - Used in the header of the website

3. **icon_group.png** - Guest icon
   - Used to represent the number of guests in place listings

4. **icon_bed.png** - Bedroom icon
   - Used to represent the number of bedrooms in place listings

5. **icon_bath.png** - Bathroom icon
   - Used to represent the number of bathrooms in place listings

## Note:
These images are referenced in the CSS/HTML files but need to be provided separately. You can:
- Download the actual AirBnB images
- Create your own custom icons
- Use placeholder images from icon libraries

For the icons (icon_group.png, icon_bed.png, icon_bath.png), recommended size is approximately 40x40 pixels.

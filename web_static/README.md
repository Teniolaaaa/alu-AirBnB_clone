# AirBnB clone - Web static

This project is the first step in building the front end of an AirBnB clone. It focuses on creating static HTML pages with CSS styling.

## Project Description

Before developing a complete web application, we build the front end step-by-step:
- Create simple HTML static pages
- Apply CSS styling
- Use a style guide
- Create fake content for prototyping
- No JavaScript
- No dynamic data

## Learning Objectives

- Understanding HTML structure and markup
- Working with the DOM (Document Object Model)
- Using HTML elements, tags, and attributes
- Applying CSS styling and selectors
- Computing CSS Specificity
- Understanding Box Model properties
- Creating W3C compliant code

## Requirements

- All files end with a new line
- Code is W3C compliant
- CSS files are in the `styles/` folder
- Images are in the `images/` folder
- No use of `!important` or `id` selectors in CSS
- No use of `img`, `embed`, or `iframe` tags
- No JavaScript

3. **Zoning done!** - Enhanced layout with logo and icons
4. **Search!** - Add filters box with search button
5. **More filters** - Add location and amenities filters
6. **It's (h)over** - Add dropdown menus on hover
7. **Display results** - Add places section with listings
8. **More details** - Add detailed information to place listings
